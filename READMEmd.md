## Что делает калькулятор

Вы вводите измерения с экрана осциллографа (координаты предельной петли и вершины частных петель), калькулятор автоматически:

- **Считает масштабы осей** m<sub>x</sub> = n₁·a<sub>x</sub>/(2π·r·R1) и m<sub>y</sub> = R2·C·a<sub>y</sub>/(S·n₂)
- **Обрабатывает предельную петлю** — H<sub>c</sub>, B<sub>0</sub>, H<sub>m</sub>, B<sub>m</sub>, μ<sub>m</sub>
- **Строит основную кривую B(H)** — по вершинам N частных петель, с tooltip-ами при наведении на точку
- **Строит кривую Столетова μ(H)** — лин. и лог. шкала по X, выделение максимума μ<sub>max</sub>
- **Считает погрешности** — настраиваемые источники (Δ отсчёта, δR, δC, δr, δS, δa), полная цепочка δm<sub>x</sub>, δm<sub>y</sub>, δH<sub>c</sub>, δB<sub>0</sub>, δH<sub>m</sub>, δB<sub>m</sub>, δμ<sub>m</sub>
- **Готовит печатные графики** — превью в стиле ВАК + генерируемый Python-скрипт `plot_vak.py`
- **Протокол** — готовый текст для вставки в отчёт (включая раздел погрешностей)

Кнопка **«★ Данные занятия»** загружает встроенные данные эксперимента от 24.04.2026 (11 точек таблицы 6.1 + предельная петля).

## Физика

### Масштабы осей осциллографа

```
m_x = n₁ · a_x / (2π · r · R1)         — А/(м·дел),  H = m_x · x
m_y = R2 · C · a_y / (S · n₂)           — Тл/дел,    B = m_y · y
```

### Предельная петля гистерезиса

С экрана осциллографа снимают 4 величины в делениях: горизонтальный размер петли через центр `2x_c`, вертикальный размер `2y_0`, координаты вершины `x_m`, `y_m`. Отсюда:

```
H_c = m_x · x_c        — коэрцитивная сила      (x_c = 2x_c/2)
B_0 = m_y · y_0        — остаточная индукция    (y_0 = 2y_0/2)
H_m = m_x · x_m        — H в вершине C
B_m = m_y · y_m        — индукция насыщения
μ_m = B_m / (μ₀ · H_m) — предельная магнитная проницаемость
```

### Основная кривая намагничивания

По вершинам N частных петель строится непрерывная кривая B(H). Для каждой точки:

```
x = 2x/2,  y = 2y/2
H = m_x · x,  B = m_y · y,  μ = B / (μ₀ · H)
```

### Кривая Столетова μ(H)

Зависимость μ от H при росте поля имеет характерный максимум μ<sub>max</sub> (соответствует области наиболее активного движения доменных стенок), затем убывает при насыщении. Сравнение μ<sub>max</sub> с μ<sub>m</sub> позволяет оценить, насколько достигнуто насыщение.

### Погрешности

```
δm_x  = √(δa_x² + δr² + δR1²)
δm_y  = √(δR2² + δC² + δa_y² + δS²)

δH_c  = √(δm_x² + δx_c²)         δx_c  = Δотсч/(2·x_c)
δB_0  = √(δm_y² + δy_0²)         δy_0  = Δотсч/(2·y_0)
δH_m  = √(δm_x² + δx_m²)         δx_m  = Δотсч/(2·x_m)
δB_m  = √(δm_y² + δy_m²)         δy_m  = Δотсч/(2·y_m)
δμ_m  = √(δB_m² + δH_m²)
```

где Δотсч — погрешность отсчёта 2x, 2y по шкале экрана (по умолчанию 0,5 дел).

## Как пользоваться

### 1. Параметры

Проверьте параметры стенда. Производные m<sub>x</sub> и m<sub>y</sub> пересчитываются автоматически и показываются в сводной таблице.

### 2. Предельная петля

С экрана осциллографа снимают 4 значения в делениях:

| Величина | Что это | Откуда |
|----------|---------|--------|
| 2x<sub>c</sub> | горизонтальный размер петли через центр | даёт H<sub>c</sub> |
| 2y<sub>0</sub> | вертикальный размер петли через центр | даёт B<sub>0</sub> |
| x<sub>m</sub> | координата вершины C по X | даёт H<sub>m</sub> |
| y<sub>m</sub> | координата вершины C по Y | даёт B<sub>m</sub>, μ<sub>m</sub> |

Кнопка **«★ Данные занятия»** загружает значения от 24.04.2026 (2x<sub>c</sub>=2.5, 2y<sub>0</sub>=4.5, x<sub>m</sub>=4.0, y<sub>m</sub>=3.0).

### 3. Основная кривая B(H)

Таблица 6.1 — координаты вершин N частных петель. По данным 2x и 2y вычисляются H, B и μ для каждой точки. Интерактивный график: наведите курсор на точку — увидите H, B, μ и номер точки. Точка C предельной петли отмечается красным ромбом. Кривая интерполирована монотонным кубическим сплайном PCHIP (плавная, но не «прострелит» точки).

### 4. Кривая μ(H)

Зависимость μ от H — кривая Столетова. Переключатель **Лин. / Лог.** меняет масштаб оси X. В сводке: μ<sub>max</sub> на основной кривой, координаты соответствующей точки, отношение μ<sub>max</sub>/μ<sub>m</sub>.

### 5. Погрешности

Настраиваемые источники погрешностей: Δотсч = 0,5 дел, δR = 5%, δC = 5%, δr = 2%, δS = 2%, δa = 3% (по умолчанию). Калькулятор показывает:
- относительные погрешности масштабов δm<sub>x</sub>, δm<sub>y</sub>
- относительные погрешности отсчёта δx<sub>c</sub>, δy<sub>0</sub>, δx<sub>m</sub>, δy<sub>m</sub>
- итоговые δH<sub>c</sub>, δB<sub>0</sub>, δH<sub>m</sub>, δB<sub>m</sub>, δμ<sub>m</sub> по формуле √(δm² + δотсч²)
- округлённую запись в виде (X ± Δ) ед. — готова для протокола

### 6. Печать графиков

Превью обоих графиков в стиле ВАК + кнопка **«↓ Скачать plot_vak.py»**.
Скрипт содержит ваши данные и готов к запуску:

## Формулы

### Масштабы

```
m_x = n₁·a_x/(2π·r·R1)
m_y = R2·C·a_y/(S·n₂)
```

### Перенос погрешностей для произведения/частного

```
δ(A·B/C) = √(δA² + δB² + δC²)
```

### PCHIP-сплайн (для гладких кривых)

Монотонный кубический сплайн Эрмита по алгоритму Фритча–Карлсона (1980). В Chart.js — встроен через `cubicInterpolationMode: 'monotone'`. В Python-скрипте — реализован руками без зависимости от scipy:

```
1. Секущие наклоны:    m_i = (y_{i+1} − y_i)/(x_{i+1} − x_i)
2. Касательные:        d_i = средневзвешенный гармонический(m_{i-1}, m_i)
                              если знак(m_{i-1})·знак(m_i) > 0
                       d_i = 0, если экстремум
3. Концы — формула Бесселя с ограничением |d| ≤ 3·|m|
4. Интерполяция кубическими полиномами Эрмита, 300 точек на кривую
```

Свойство монотонности гарантирует, что кривая не «прострелит» точки — критично для экспериментальных данных.
