Параметры
Исходные данные со стенда установки. Кольцевой ферритовый сердечник с двумя обмотками; осциллограф работает в режиме фигуры B(H).
ℹ На вход X осциллографа подаётся напряжение с резистора R1, пропорциональное току первичной обмотки и, следовательно, напряжённости H. Ключевые формулы
Напряжённость магнитного поля — из тока первичной обмотки
H = mx · x,   mx = n₁·ax / (2π·r·R1)
x — горизонтальное отклонение луча в делениях. mx — масштаб оси X в А/(м·дел).
Магнитная индукция — из ЭДС вторичной обмотки через RC-интегратор
B = my · y,   my = R2·C·ay / (S·n₂)
y — вертикальное отклонение луча в делениях. my — масштаб оси Y в Тл/дел. RC-цепочка интегрирует наведённую ЭДС, так что напряжение на C пропорционально B.
Магнитная проницаемость
μ = B / (μ₀·H),   μ₀ = 4π·10⁻⁷ Гн/м
Безразмерная. Предельная μm вычисляется по координатам точки C (вершина петли): μm = Bm/(μ₀·Hm).
Параметры предельной петли гистерезиса
Bm — индукция насыщения (вершина петли)
B0 — остаточная индукция (при H = 0)
Hc — коэрцитивная сила (при B = 0)
Измеряются вдвое: 2xc, 2y0 — расстояния через центр петли; xm, ym — координаты вершины C.
Предельная петля гистерезиса
Измеренные на экране осциллографа координаты предельной петли. Все размеры — в делениях сетки экрана.
ℹ На осях, проходящих через центр петли, измеряются 2xc (горизонтальный размер по оси B = 0) и 2y0 (вертикальный размер по оси H = 0). Это даёт Hc и B0. Координаты вершины петли C — это xm, ym; они дают Hm и Bm.
Координаты предельной петли
⇒ Hc = mx · xc, где xc = 2xc / 2
⇒ B0 = my · y0, где y0 = 2y0 / 2
⇒ Hm = mx · xm
⇒ Bm = my · ym

Основная кривая намагничивания B(H)
Снимается ступенями: уменьшая ток в первичной обмотке, измеряем расстояния 2x и 2y между вершинами частных петель. Кривая — линия через эти вершины.
ℹ Каждая строка таблицы — это одна частная петля. По данным 2x и 2y вычисляются H = mx·x и B = my·y, где x = 2x/2, y = 2y/2. По полученным точкам строится основная кривая.

Зависимость μ(H) — кривая Столетова
Магнитная проницаемость μ = B/(μ₀·H), вычисленная по точкам основной кривой намагничивания.
ℹ При росте H проницаемость μ сначала возрастает, достигает максимума, а затем при насыщении быстро убывает. Максимум μ в делах от μm и есть предельная магнитная проницаемость, оцениваемая по координатам Hm, Bm точки C предельной петли.

Оценка погрешностей
Расчёт абсолютных и относительных погрешностей итоговых величин по правилу переноса погрешностей для функций произведения/частного.
ℹ Погрешности измерений обусловлены: инструментальной погрешностью отсчёта по сетке осциллографа Δ(2x) = Δ(2y) = 0,5 дел; погрешностями номиналов элементов схемы (резисторы 5%, конденсатор 5%, витки ±1, размеры ±1…2%). Для половинных величин (xc = 2xc/2 и т.д.) Δxc = Δy0 = Δxm = Δym = 0,25 дел.

Относительные погрешности масштабов
Масштаб H по оси X
δmx = √(δax² + δr² + δR1²)
mx = n₁·ax/(2π·r·R1) — погрешности витков пренебрежимы.
Масштаб B по оси Y
δmy = √(δR2² + δC² + δay² + δS²)
my = R2·C·ay/(S·n₂).
